# Project Name
**PET CARE IN WINTER**


## Purpose
This project is built to provide a simple and secure user authentication system using Firebase. Users can register with email and password, log in, reset their password, or sign in using Google. Once logged in, users can view and update their profile information, including their display name and profile picture. The project focuses on a responsive design, smooth user experience, and proper error handling for login and registration processes.


## Live URL
[Live Project Link Here]((https://assignment-09-by-meem.netlify.app/))

## Key Features
- Responsive design for mobile, tablet, and desktop
- User authentication (Login/Registration) using Firebase
- Password reset functionality
- Google Sign-in integration
- Dynamic content loading using React Router
- Smooth animations using AOS
- Toast notifications for user actions
- Secure handling of user data
